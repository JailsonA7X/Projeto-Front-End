👟 Digital Store - Front-end Project
Este é um projeto de e-commerce completo desenvolvido para o módulo de Front-end da Digital College. O foco principal foi a criação de uma interface moderna, fiel ao design do Figma e totalmente responsiva.

🚀 Tecnologias Utilizadas
React.js (Vite): Para uma estrutura de componentes rápida e moderna.

Tailwind CSS: Framework utilizado para toda a estilização e responsividade.

Lucide React: Biblioteca de ícones para o Header, Footer e Redes Sociais.

✨ Funcionalidades e Diferenciais
Internacionalização: Interface totalmente traduzida para o inglês para um visual profissional.

Layout Responsivo: O site se adapta de monitores UltraWide até dispositivos mobile.

Social Media: Ícones interativos do Facebook, Instagram e X (Twitter) no rodapé.

Vitrine Dinâmica: Seção de produtos e coleções em destaque com efeitos de hover.

🛠️ Como rodar o projeto
Clone este repositório.

No terminal, instale as dependências:

Bash
npm install
Inicie o servidor de desenvolvimento:

Bash
npm run dev
Acesse o link gerado (geralmente http://localhost:5173/).
